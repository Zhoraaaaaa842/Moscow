# 🗺️ Moscow Walk

Интерактивная карта Москвы (в пределах МКАД) — выбирай случайный район, получай список интересных мест и отмечай посещённые районы.

## Стек
- [Tauri v2](https://v2.tauri.app/) — десктопная оболочка
- Vanilla JS + HTML/CSS — фронтенд
- Leaflet.js — интерактивная карта
- OpenStreetMap / Overpass API — данные о районах и POI

## Быстрый старт

```bash
# Установить зависимости
npm install

# Запустить в режиме разработки
npm run tauri dev

# Собрать приложение
npm run tauri build
```

## Требования
- Node.js 18+
- Rust (stable)
- [Tauri prerequisites](https://v2.tauri.app/start/prerequisites/)

## Функциональность
- 🎲 Случайный выбор района
- 📍 Топ интересных мест района (парки, музеи, кафе, достопримечательности)
- ✅ Отметить район как посещённый — он закрашивается на карте
- 💾 Прогресс сохраняется локально
- 🌙 Тёмная тема
